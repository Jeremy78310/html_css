    • Propriétés CSS utilisées dans cette page

    Decriver leur usage et leurs valeurs possibles

    1. background : La propriété background est une propriété raccourcie qui permet de définir les différentes valeurs des propriétés liées à la gestion des arrière-plans d'un élément.
    2. color : Le type de données CSS color permet de représenter des couleurs dans l'espace de couleurs RGB. Une couleur pourra être décrite de trois façons : grâce à un mot-clé (comme blue ou transparent par exemple) en utilisant le système de coordonnées cubiques RGB (grâce à la notation #-hexadecimal ou aux notations fonctionnelles rgb() et rgba() en utilisant le système de coordonnées cylindriques HSL (grâce aux notations fonctionnelles hsl() et hsla() En plus de la couleur exprimée dans l'espace RGB, une valeur color contient également un canal alpha qui décrit la transparence de l'image et donc la façon dont cette image se compose avec son arrière-plan.
    3. float : La propriété float indique qu'un élément doit être retiré du flux normal et doit être placé sur le côté droit ou sur le côté gauche de son conteneur.
    4. font-family : Pour changer la police d'écriture, ex arial ...
    5. font-size : Pour changer la taille de la police
    6. height : Pour changer la hauteur
    7. left : Position à partir du bord gauche de l'écran, en px ou autre.
    8. line-height : Change la hauteur entre les lignes.
    9. margin : Change la tailles des marges exterieurs.
    10. padding : Change la tailles des marges interieurs.
    11. postion : Positionne un élément, ex fixed, absolu, relatives... S'utilise conjointement avec left, right, top, bottom
    12. text-align : Alignement du texte right,left, center, justify
    13. text décoration : Dans notre cas on l'a utilisé pour retirer l'und
    14. top :Position à partir du haut de l'écran.
    15. transition : La propriété transition est une proprété raccourcie pour les propriétés : transition-propery, duration, timming-function, delay.
    16. width : Définit la largeur d'un élément.

    Propriétés CSS à rechercher

    Rechereché les propriétés ci dessous, faites des tests sur cette page en utilisant la console de débuggage pour tester les valeurs qu'elles peuvent prendre (F12 dans le navigateur)

    1. font-weight : La propriété CSS font-weight permet de définir la graisse utilisée pour le texte. Les niveaux de graisse disponibles dépendent de la police (cf. font-family). Certaines fontes n'existent qu'avec les niveaux de graisses normal et bold lighter on peut meme utilisr des valeur dde 1 à 1000 meme avec des virgules.
    2. letter-spacing : La propriété letter-spacing définit l'interlettre utilisée pour les caractères qui composent le texte.
    3. word-spacing : La propriété word-spacing définit la règle d'espacement utilisée entre les balises et entre les mots.
    4. text-shadow : La propriété text-shadow ajoute des ombres au texte. Elle accepte une liste d'ombres à appliquer au texte et aux décorations de l'élément. Chaque ombre est décrite par une certaine combinaison de décalages X et Y de l'élément, de rayon de flou et de couleur.
    5. background-image : La propriété background-image permet de définir une ou plusieurs images comme arrière(s)-plan(s) pour un élément.
    6. background-size : La propriété CSS background-size définit la taille des images d'arrière-plan pour l'élément. La taille de l'image peut être contrainte, complètement ou partiellement afin de conserver ses proportions.
    7. opacitiy : La propriété opacity définit la transparence d'un élément. Autrement dit, elle permet de définir le degré de visibilité de l'arrière-plan sur lequel est placé l'élément.
    8. border-style : La propriété CSS border-style est une propriété raccourcie qui permet de définir le style des lignes utilisées pour les bordures des quatre côtés de la boîte d'un élément.
    9. box-shadow : La propriété CSS box-shadow ajoute des ombres sur une balise par des virgules. Une boîte d'ombre est définie avec des décalages horizontal X et vertical Y par rapport à l'élément, avec des rayons de flou et d'étalement et avec une couleur.
    10. list-style-image : La propriété list-style-image définit l'image utilisée comme puce devant les éléments de listes. On peut également utiliser la propriété raccourcie list-style.
    11. border : La propriété CSS border est une propriété raccourcie qui permet de définir les propriétés liées à la bordure. border peut être utilisée pour définir les valeurs de border-width, border-style et border-color.
    12. border-radius : La propriété border-radius permet de définir des coins arrondis pour la bordure d'un élément. La courbure de chaque coin est définie avec un ou deux rayons de courbures qui permettent de définir un arc de cercle ou un arc d'ellipse.
    13. border-with : La propriété CSS border-width est une propriété raccourcie qui définit l'épaisseur de la bordure d'un élément. Cette propriété raccourcie définit les propriétés détaillées border-top-width, border-right-width, border-bottom-width border-left-width. Si on utilise les propriétés logiques, elle définit border-block-start-width, border-block-end-width, border-inline-start-width et border-inline-end-width. Afin de paramétrer une bordure de façon plus pratique, on pourra utiliser la propriété raccourcie border.
    14. font-variant : La propriété font-variant est une propriété raccourcie représentant les propriétés : font-variant-caps, font-variant-numeric, font-variant-alternates, font-variant-ligatures et font-variant-east-asian. On peut aussi utiliser les valeurs de la propriété font-variant définies avec CSS Level 2 font-variant, en utilisant la propriété raccourcie font.
    15. background-color : La propriété background-color permet de définir la couleur utilisée pour l'arrière-plan d'un élément (celle-ci peut être une couleur transparente).
    16. !important : A mettre à la fin d'une ligne de CSS pour forcer le navigateur à appliquer le style, il ne sera pas surcharger et il faut éviter de l'utiliser.
